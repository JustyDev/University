# Лабораторная работа 1. Графический редактор

Приложение написано на Python с использованием стандартной библиотеки `tkinter`.

## Запуск на macOS

```bash
cd "/Users/justy-dev/Documents/University/Semester-6/курсовая/code"
python3 main.py
```

## Возможности

- построение отрезков в режиме тянущейся линии;
- построение окружностей от центра к точке на окружности;
- построение дуг по трём точкам;
- построение полилиний из чередующихся отрезков и дуг по трём точкам;
- выбор одиночных объектов мышью;
- выбор прямоугольной рамкой по полному попаданию;
- выбор прямоугольной рамкой по пересечению;
- удаление выбранных объектов и отдельный режим удаления;
- настройка типа и цвета заливки для замкнутых контуров;
- сохранение и открытие рисунков в JSON-файлах.

## Структура проекта

```text
code/
├── main.py              # точка входа
└── editor/
    ├── app.py           # главное окно, меню, режимы работы и события мыши
    ├── config.py        # общие константы интерфейса
    ├── dialogs.py       # диалог выбора типа и цвета заливки
    ├── geometry.py      # геометрические расчёты: дуги, окружности, рамки
    └── models.py        # модель графического объекта и отрисовка на Canvas
```

## Управление

- `Файл -> Открыть/Сохранить` - работа с файлами.
- `Режим` - выбор текущего режима работы.
- `Правка -> Заливка выбранных контуров` - изменение заливки окружностей и замкнутых полилиний.
- `Backspace` или `Delete` - удалить выбранные объекты.
- `Escape` - отменить текущий ввод.
- Двойной щелчок завершает построение полилинии.
